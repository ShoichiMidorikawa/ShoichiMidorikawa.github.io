今までやったことの確認として最終的に全体のファイル配置やコードなどがどうなっていれば良いのか確認していきましょう。

次に『Page1.js』ファイルの確認です。最終的に下記のようになっていれば大丈夫です。
※ 入力したコードの順番が少し違っていても指定された通りに入力していれば問題ありません。


$(document).ready(function(){
 $('.bxslider').bxSlider({
	auto: true,
	mode:'fade',
	pause:4000,
    });
});

$(function() {
  $('#main-menu').smartmenus();
});

$(document).ready(function(){
$("#fitvids").fitVids();
});

次に『Page1.css』ファイルの確認です。最終的に下記のようになっていれば大丈夫です。
※ 入力したコードの順番が少し違っていても指定された通りに入力していれば問題ありません。


ul.bxslider ,
ul.bxslider li {
margin:0;
padding:0;
}

.fitvids {
position: relative;
margin: 20px 0;
padding-bottom: 56.25%;
padding-top: 30px;
overflow: hidden;
max-width: 100%;
height: auto;
}
 
.fitvids iframe,
.fitvids object,
.fitvids embed {
position: absolute;
top: 0;
left: 0;
width: 100%;
height: 100%;
}

p.resizeimage img { width: 100%; }
//ウインドウ幅に合わせて画像サイズを変化させる

これでサイトを作ることができたと思います。

もし、メニューやスライドショーがきちんと動作していない場合は下記の可能性があるので確認してみてください。

  • 『 . 』と『 , 』の部分の入力ミス。

  • 『 ; 』と『 : 』の部分の入力ミス。

  • 全角のスペースが入っている。

  • リンクの入力ミス。
    →ファイルを置いている場所のURLの入力ミス。

    • 例：src="Photo/画像1.jpg"→src="../Photo/画像1.jpg"
    →Yahoo!などのURLの入力ミス。
    →href="http://www.yahoo.co.jp"→href="http://www.yahoo.co.jp/
